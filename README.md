# gh-runner-devops-team-2

This is the final project of team 2

You need to install Vagrant and Docker.

## .env file:

```
  RUNNER_URL=""
  GITHUB_URL=""
  GITHUB_HASH=""
  RUNNER_DIR=""
  GITHUB_TOKEN=""
  RUNNER_VERSION=""
  BOX=""
```

## Services

```
Jenkins (port 8081)
Nexus (port 8082)
Portainer (port 8083)
Sonarqube (port 8084)
```
